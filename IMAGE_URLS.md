# Image URLs Reference

This file lists all the image placeholders in the website that need to be updated with actual URLs from priesmont.com.

## How to Find Image URLs

1. Visit https://priesmont.com/
2. Right-click on any image you want to use
3. Select "Copy image address" or "Copy image URL"
4. Replace the placeholder URLs in `index.html` with the actual URLs

## Alternative: Using Facebook Images

If you prefer to use images from the Facebook page:
1. Visit https://www.facebook.com/profile.php?id=61584225191225
2. Find the images you want
3. Right-click and copy the image URL
4. Replace the placeholders in `index.html`

## Image Requirements

- **Hero image**: Minimum 1920x1080px (wide format recommended)
- **About image**: Square format (1:1 aspect ratio) recommended, minimum 800x800px
- **Gallery images**: 4:3 aspect ratio recommended, minimum 1200x900px

## Note

All images have fallback placeholders that will show if the URLs don't work. Once you update the URLs with actual image addresses, the placeholders will be replaced automatically.
